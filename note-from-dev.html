Note from the Developer

Unique Features & Highlights

  • Modern Responsive UI: Built with Tailwind CSS for a clean, vibrant, and fully responsive experience on all devices.
  • Consistent Branding: All references to QUTRIX and Gobi Arts & Science College are uniform and accurate across every page and section.
  • Single-Row Navbar: Logo, college info, and hamburger menu are always in a single row, perfectly aligned on all screen sizes.
  • Sticky Footer: Footer always stays at the bottom, even with little content, for a professional look.
  • Event Modals: Interactive event modals with rules and action buttons on the Events page.
  • Dropdown Info Tablets: About page features modern dropdowns for college, department, GAIT, QUTRIX, and GAITFEST, with only one open at a time.
  • Mobile-First Navigation: Hamburger menu and mobile nav are smooth, accessible, and consistent with desktop.
  • Contact Info: Clear, consistent contact information in the footer of every page.
  • Developer Credit: "~Vibe Coded by Mr. Mathiraja~" is present in every footer as a signature touch.
  • Accessibility: Uses semantic HTML, proper aria-labels, and keyboard navigation for modals and menus.

Thank you for exploring QUTRIX 2025! This project is a labor of love, designed to set a new standard for college event websites. If you have feedback or want to collaborate, feel free to reach out.
